HTML

Ao contrário em que muitos pensão o HTML não é considerado linguagem de programação, mas sim “linguagem de marcação”. Essa “linguagem de marcação” foi criada em 1991, por TIM Berners-Lee, pois antes as empresas compartilhavam documentos entre si, mas não tinha a possibilidade de fazer de forma rápida como hoje. Essa criação não viria apenas para facilitar a vidas das empresas, mas mudaria o mundo.

A abreviação HTML vem do inglês Hyper Text Markup Language, que significa “Linguagem de Marcação de Hipertexto”. Olhe para essa página que está vendo esse artigo, nela existe: Cabeçalho, Título, Parágrafos, Imagem, Rodapé e muitos outros elementos. Toda essa página é escrita em HTML, onde são usadas as tags (etiquetas) para sinalizar onde será implementado cada elemento. Com a junção de várias tags é feito todo o esqueleto da página, mas tudo isso não é feito apenas pelo HTML, junto com ele vem o CSS, que será nosso próximo tópico.

CSS

Assim como o HTML não é considerada uma linguagem de programação, o CSS é uma “linguagem de estilos”. A abreviação CSS vem do também do inglês Cascading Style Sheet “Folha de Estilo em Cascata, essa tecnologia vem para separar a parte estrutural que é feito pelo HTML da parte estética.

Com o CSS podemos estilizar as tags, classes e outros identificadores. Alteramos cor, fonte, tamanho, posicionamento dos elementos como fotos, vídeos, textos e muito mais. Nesse determinamos, o valor da borda, margem, altura, preenchimento. Importante ressaltar que podemos usar mais de um arquivo CSS ao mesmo tempo, daí vem a palavra “cascata”. Permitindo várias interações e estilizações, mas precisamos tomar cuidado para não se perder no código e deixá-lo confuso para quem for ler.
